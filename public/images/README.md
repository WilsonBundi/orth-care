# Images Directory

## How to Add Images

### Required Images

1. **Hero Images** (1920x1080px)
   - `hero-orthopedic-care.jpg` - Main homepage hero
   - `hero-consultation.jpg` - Alternative hero image

2. **Service Images** (600x400px)
   - `service-orthopedics.jpg`
   - `service-sports-medicine.jpg`
   - `service-physical-therapy.jpg`
   - `service-joint-replacement.jpg`

3. **Team Photos** (400x400px)
   - `team-doctor-1.jpg`
   - `team-nurse-1.jpg`
   - `team-staff.jpg`

## Where to Get Images

### Free Professional Healthcare Images:

1. **Unsplash** - https://unsplash.com/s/photos/healthcare
   - Search: "doctor", "orthopedic", "medical consultation"
   - Free to use, high quality

2. **Pexels** - https://www.pexels.com/search/medical/
   - Search: "healthcare", "hospital", "doctor patient"
   - Free for commercial use

3. **Pixabay** - https://pixabay.com/images/search/healthcare/
   - Search: "medical", "doctor", "hospital"
   - Free, no attribution required

## Image Optimization

Before adding images:
1. Resize to recommended dimensions
2. Compress using TinyPNG.com
3. Save as JPG (quality 80-85%)
4. Keep file size under 200KB
