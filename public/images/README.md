# Medical Images

Please place your medical images in this folder with the following names:

1. `doctor-consultation.jpg` - Doctor with patient consultation
2. `medical-facility.jpg` - Modern medical facility/clinic
3. `digital-health.jpg` - Digital health technology/tablet
4. `joint-pain.jpg` - Joint/bone pain illustration (ankle/knee)
5. `shoulder-pain.jpg` - Shoulder/elbow pain illustration
6. `knee-pain.jpg` - Knee pain illustration (running figure)

These images will be used throughout the landing page to showcase orthopedic care services.

## Image Requirements
- Format: JPG, PNG, or WebP
- Recommended size: 800x600px or larger
- Aspect ratio: 4:3 or 16:9
- File size: Optimized for web (under 500KB each)
